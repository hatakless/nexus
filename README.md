# Nexus — Мессенджер

Современный веб-мессенджер с дизайном уровня Telegram/Discord.

## Быстрый старт

### 1. Установка зависимостей
```bash
pip install -r requirements.txt
```

### 2. Запуск
```bash
python main.py
```

### 3. Открыть в браузере
```
http://localhost:8000
```

## Структура проекта

```
nexus/
├── main.py          — FastAPI приложение + WebSocket эндпоинт
├── models.py        — Модели базы данных (SQLAlchemy)
├── auth.py          — Регистрация, вход, токены
├── messages.py      — Чаты, сообщения, реакции, файлы
├── ws.py            — WebSocket менеджер
├── database.py      — Подключение к SQLite
├── requirements.txt — Зависимости
├── static/
│   ├── auth.html    — Страница входа/регистрации
│   └── app.html     — Главный экран мессенджера
└── uploads/         — Загруженные файлы и аватары
```

## Возможности

- Регистрация и вход
- Личные чаты и групповые чаты
- Мгновенная доставка сообщений через WebSocket
- Индикатор «печатает...»
- Реакции на сообщения
- Прикрепление файлов и изображений
- Профиль пользователя с аватаром
- Онлайн-статусы
- Система достижений
- Поиск пользователей
- Тёмная тема (glassmorphism дизайн)
